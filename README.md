# Guess My Number

A simple number guessing game built with HTML, CSS and Javascript.
The player has to guess a randomly generated number between 1 and 20 while trying to achieve the highest possible score.

## Features

- Random number generation;
- Score tracking;
- High score;
- Restart game;
- Input validation.

## Technologies

- HTML5;
- CSS3;
- JavaScript.

## How to Play

1. Enter a number between **1 and 20**.
2. Click **Check!**
3. Receive feedback:
   - Too high!
   - Too low!
   - Correct number!
4. Try to finish with the highest score possible.
